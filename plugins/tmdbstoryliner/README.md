# TMDB剧情简介更新器

## 简介

TMDB剧情简介更新器是一个MoviePilot插件，可以定时从TMDB获取剧集和电影的剧情简介，并将英文内容翻译成中文。

## 功能特点

- 定时任务：支持cron表达式的定时执行
- 多媒体支持：支持电影和电视剧的剧情简介更新
- 翻译服务：集成百度翻译API，将英文剧情简介翻译成中文
- 配置化：提供友好的Web配置界面
- 历史记录：记录更新历史，方便查看更新状态

## 安装说明

1. 在MoviePilot插件市场中搜索"TMDB剧情简介更新器"
2. 点击安装并重启MoviePilot服务
3. 在插件设置中配置相关参数

## 配置说明

### TMDB API密钥

1. 访问[TMDB官网](https://www.themoviedb.org/settings/api)注册账号
2. 申请API密钥

### 百度翻译API密钥

1. 访问[百度翻译开放平台](https://fanyi-api.baidu.com/)注册账号
2. 创建应用获取APP ID和密钥

## 使用说明

1. 启用插件
2. 配置TMDB API密钥和百度翻译API密钥
3. 设置执行周期（cron表达式）
4. 选择需要更新的媒体类型（电影/电视剧）
5. 插件将按设定的时间自动更新剧情简介

## 注意事项

- 需要网络连接以访问TMDB和百度翻译API
- 请确保API密钥正确配置
- 翻译服务可能产生费用，请参考对应服务商的收费标准
